# hashtable

First term Programming Coursework for SPBGMTU, Theme 8: Hashtable implementation in C++

The pdf can be built from source in a linux environment by running `make` in `paper/`, required dependencies:
- C++ compiler
- cmake, make
- typst compiler
- plantUML (+graphviz dot)
- python venv with pandas, matplotlib, seaborn

The benchmarks are quite long to run because of high repetitions and lots of different sizes (used to generate detailed plots), you can change them in `bench.cpp` (and yes they are hardcoded)
